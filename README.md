# Лабораторная работа №7: Адаптивная верстка с Flexbox

## Цель работы
Освоить современные технологии CSS-разметки — Flexbox для создания адаптивных веб-страниц.

## Описание
В ходе лабораторной работы изучаются основные свойства Flexbox:
- `flex-direction` (направление элементов)
- `justify-content` (выравнивание по главной оси)
- `align-items` (выравнивание по поперечной оси)
- `flex-wrap` (перенос элементов)
- `gap` (отступы между элементами)
- `flex-grow`, `flex-shrink`, `flex-basis` (размеры элементов)

## Файлы проекта
1. `index.html` - Основная страница с демонстрацией Flexbox
2. `styles.css` - Стили для демонстрации свойств Flexbox
3. `flexbox.html` - Практические примеры использования Flexbox
4. `flexbox.css` - Стили для практических примеров
5. `README.md` - Описание лабораторной работы
6. `img/` - Папка со скриншотами

## Практические примеры
1. **Навигационное меню** - горизонтальное меню с равномерным распределением
2. **Карточки товаров** - адаптивные карточки с переносом на новую строку
3. **Центрирование элемента** - вертикальное и горизонтальное центрирование

## Как запустить
1. Откройте файл `index.html` в браузере
2. Для просмотра практических примеров откройте `flexbox.html`
3. Используйте Live Server в VS Code для автоматического обновления
